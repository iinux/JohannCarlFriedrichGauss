移动端链接字体大小调整方法

方法1：固定像素大小 (px)

固定大小链接 (20px) 另一个固定大小链接
注意：固定大小在不同设备上显示效果一致，但可能无法适应所有屏幕尺寸。
a { font-size: 20px; }

方法2：相对单位 (em)

相对大小链接 (1.2em) 另一个相对大小链接
注意：em单位相对于父元素的字体大小，适合需要相对于上下文调整大小的场景。
.parent { font-size: 18px; }
a { font-size: 1.2em; } /* 计算后为21.6px */

方法3：响应式单位 (rem)

响应式链接 (1.2rem) 另一个响应式链接
注意：rem单位相对于根元素(html)的字体大小，适合创建整体可缩放的界面。
html { font-size: 16px; }
a { font-size: 1.2rem; } /* 计算后为19.2px */
@media (max-width: 480px) {
  html { font-size: 18px; }
}

方法4：视口单位 (vw)

视口单位链接 (5vw) 另一个视口单位链接
注意：vw单位相对于视口宽度，字体大小会随屏幕宽度变化，适合需要完全响应式的设计。
a { font-size: 5vw; } /* 视口宽度的5% */

方法5：媒体查询

媒体查询链接 另一个媒体查询链接
注意：媒体查询允许在不同屏幕尺寸上应用不同的样式，提供最精确的控制。
a { font-size: 16px; }
@media (min-width: 480px) {
  a { font-size: 18px; }
}
@media (min-width: 768px) {
  a { font-size: 20px; }
}
